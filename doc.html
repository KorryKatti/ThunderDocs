Publish Apps

Learn how to Publish Apps to the thunder store , you only need a github account.

Name Description Requirement
Github Account Needed to submit applications of your app , is auto-approved true
Python Only Is recommended but not necessary.Instrcutions for non python apps ahead false
Thunder account You need one to use community features, imageboard doesn't require it false

Keeping all that in mind you can now publish your apps.Follow the instructions given at thapps repo or you could fill out this form which is available here.Following the thapps repo is a faster way for your app to get published.Forms method may cause you not being able to handle the version control of your published app due to the nature of the bot which approves the apps. Altho we don't scan the code contents ourselves , a good number of reports to your app can result in it being taken down. Reports include malicious code and inappropriate content.

Linux

Thunder was developed on linux ( Arch Linux more specifically ) so it should work without any issues. However you might face one error on the first run that is tk not being installed.Here is guide to install that on all major linux OS

  • Ubuntu/Debian: 
    sudo apt-get install tk
  • Fedora: 
    sudo dnf install tk
  • CentOS/RHEL: 
    sudo yum install tk
  • Arch Linux: 
    sudo pacman -S tk

After installing it for your preferred linux distro you can run this code to download and run Thunder. 

                
                  $  git clone https://github.com/KorryKatti/Thunder && cd Thunder && ./run.sh
Does not require any admin privileges.Or you can download the .appimage file from here : Download. Using the Appimage file is preferred as default method of installation requires to navigate to the directory everytime you wish to run Thunder.
